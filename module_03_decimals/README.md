# Module 3: Decimals

## Overview
Module 3 covers the fundamental concepts of decimals and their real-world applications. Daniel will learn how fractions and decimals represent the same values, and practice solving word problems involving money and measurements.

## Lessons in This Module

### Lesson 27: Fraction and Decimal Connection
Learn how fractions and decimals show the same value, just in different ways.

**Key Topics:**
- Converting fractions (denominators 10, 100) to decimals
- Converting decimals to fractions
- Visual models (number lines, grids, base-10 blocks)
- Comparing decimals

**Files:**
- `lesson_27_fraction_decimal_connection/concepts.md` - Core vocabulary and definitions
- `lesson_27_fraction_decimal_connection/explanations.md` - Detailed step-by-step explanations
- `lesson_27_fraction_decimal_connection/exercises.md` - 30+ practice problems
- `lesson_27_fraction_decimal_connection/quiz.json` - 5-question quick assessment
- `lesson_27_fraction_decimal_connection/solutions.md` - Complete answers with explanations

---

### Lesson 28: Decimal Word Problems
Solve real-world problems involving money, measurements, and multi-step operations.

**Key Topics:**
- Money problems (addition, subtraction, making change)
- Measurement applications
- Multi-step problem solving
- Estimation strategies

**Files:**
- `lesson_28_decimal_word_problems/concepts.md` - Real-world applications overview
- `lesson_28_decimal_word_problems/explanations.md` - Detailed problem-solving strategies
- `lesson_28_decimal_word_problems/exercises.md` - 35+ word problems
- `lesson_28_decimal_word_problems/quiz.json` - 5-question application quiz
- `lesson_28_decimal_word_problems/solutions.md` - Detailed solutions and answers

---

## Learning Path

1. Start with **Lesson 27** to understand the fraction-decimal connection
2. Practice converting in both directions
3. Move to **Lesson 28** to apply decimals in real-world contexts
4. Use the quiz files to assess understanding
5. Reference solutions to check work and learn from mistakes

## Age Appropriateness

All materials are designed for a 10-year-old learner with:
- Clear, encouraging explanations
- Relatable examples (pizza, money, sports)
- Visual models and diagrams
- Progressive difficulty
- Plenty of practice opportunities

## Time Estimates

- **Lesson 27:** 2-3 hours (concepts + exercises)
- **Lesson 28:** 2-3 hours (applications + problems)
- **Total Module Time:** 4-6 hours over 2-3 weeks
