# Module 4: Measurement

## Overview
Module 4 introduces Daniel to the metric system and real-world measurement concepts. He will learn to measure length, mass, capacity, and time using standard metric units and solve practical problems.

## Lessons in This Module

### Lesson 29: Length and Distance
Master metric units of length and convert between them.

**Key Topics:**
- Metric units: mm, cm, m, km
- Converting between units
- Measuring accurately with rulers
- Comparing distances

**Files:**
- `lesson_29_length_distance/concepts.md` - Metric units and vocabulary
- `lesson_29_length_distance/explanations.md` - Detailed conversion strategies
- `lesson_29_length_distance/exercises.md` - 75+ conversion and measurement problems
- `lesson_29_length_distance/quiz.json` - 5-question conversion quiz
- `lesson_29_length_distance/solutions.md` - Complete answers with explanations

---

### Lesson 30: Mass and Capacity
Understand weight and how much containers hold.

**Key Topics:**
- Mass: grams and kilograms
- Capacity: milliliters and liters
- The metric relationship (1 kg water = 1 L)
- Real-world applications (cooking, nutrition, medicine)

**Files:**
- `lesson_30_mass_capacity/concepts.md` - Units and definitions
- `lesson_30_mass_capacity/explanations.md` - Detailed explanations with examples
- `lesson_30_mass_capacity/exercises.md` - 80+ practice problems
- `lesson_30_mass_capacity/quiz.json` - 5-question application quiz
- `lesson_30_mass_capacity/solutions.md` - Complete solutions

---

### Lesson 31: Time
Learn to tell time and calculate elapsed time.

**Key Topics:**
- Reading analog and digital clocks
- Telling time in different formats
- Converting between time units (seconds, minutes, hours)
- Calculating elapsed time
- Solving time word problems

**Files:**
- `lesson_31_time/concepts.md` - Clock types and time units
- `lesson_31_time/explanations.md` - Reading clocks and solving time problems
- `lesson_31_time/exercises.md` - 75+ time practice problems
- `lesson_31_time/quiz.json` - 5-question time quiz
- `lesson_31_time/solutions.md` - Complete answers with strategies

## Learning Path

1. **Lesson 29** - Start with length (most intuitive metric concept)
2. **Lesson 30** - Continue with mass and capacity
3. **Lesson 31** - Conclude with time measurement
4. Use quizzes after each lesson to check understanding
5. Reference solutions to learn problem-solving approaches

## Module Features

- **Age-Appropriate**: Written for a 10-year-old with encouraging tone
- **Visual Examples**: Real-world items and comparisons
- **Progressive Difficulty**: Starting simple, building complexity
- **Comprehensive Practice**: 200+ total exercises across all lessons
- **Detailed Solutions**: Every answer explained thoroughly

## Connections to Real Life

Students learn to:
- Measure their height and arm span
- Understand food labels
- Calculate cooking times
- Tell time accurately
- Estimate distances and weights

## Time Estimates

- **Lesson 29:** 2-3 hours (conversions are key)
- **Lesson 30:** 2-3 hours (multiple unit types)
- **Lesson 31:** 2-3 hours (practice is essential)
- **Total Module Time:** 6-9 hours over 3-4 weeks
